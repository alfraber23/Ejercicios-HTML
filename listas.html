Conceptos de programación:

Función
Es un bloque de código que ejecuta una tarea particular. Recibe valores(o referencias) y los utiliza para dicha tarea.
let
Es una palabra reservada de JavaScript que se usa para definir variables en el ámbito donde se usa(es decir, locales) si intentamos hacerlo más de una vez con la misma variable aparece un error.
const
Es una palabra reservada de JavaScript que se usa para definir variables con un valor fijo, que no cambiará después en el programa. Si intentamos cambiar ese valor, obtendremos un error como respuesta.
<nav>
Este elemento es para designar una sección de una página web como una especie de "índice", es decir, una lista con enlaces que apuntan a otros lugares del mismo sitio web.
<audio>
Es para insertar un audio
<div>
Es una etiqueta que se usa como contenedor para despues poder aplicar estilos
